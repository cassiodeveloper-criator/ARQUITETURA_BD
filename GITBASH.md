                                                Git Bash

•cd = entrar
==>cd documents = entrar na pasta de documents

•ls = visualiza os documentos dentro da pasta

•git init = iniciar o repositório

•git add . = prepara os arquivos para mandar ao repositório

•git commit -m = mensagem pro repositório

•git branch -M main = falar que o repositório é o principal

•git remote add origin + codigo do repositório = vincular o repositório .git

•git push -u origin main = empurrar o arquivo pro github

•code . = abrir o vscode

•code .. = volta para a pasta anterior

•git clone = clonar o repositório

•git rm --cached + nome da pasta clonada = remover informações do antigo usuário
