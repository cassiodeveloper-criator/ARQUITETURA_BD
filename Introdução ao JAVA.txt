Introdução ao JAVA

Todo programa JAVA precisa estar dentro de uma CLASSE. Dentro dessa classe, precisamos de um método principal chamado main, que é o ponto de partida da execução.
Em JAVA, todo comando deve terminar com ponto e vírgula(;) e o código é case-sensitive - ou seja, diferencia maiúsculas de minúsculas.

Seu Primeiro Programa Java

//Todo código Java vive dentro de uma classe
public class OlaMundo {

  // O método main é onde o programa começa
   public static void main(String[] args){
    // System.out.println imprime texto na tela
    System.out.println("Olá, Java!");
  
  }

}

•public class = define a classe pública que contém seu código.
•main method = ponto de partida obrigatóriodo programa.
•System.out.println = comando para exibir mensagem na tela.

Java: Uma linguagem fortemente tipada

O que isso significa?
 Em Java, voce deve declarar o tipo de dado que uma variável vai guardar antes de usá-la. Isso inclui números, texto,valores verdadeiro/falso e muito mais.
 Essa características ajuda a prevenir erros a prevenir erros e tornar seu código mais seguro.

Tipos de dados primitivos

•int = Números inteiros sem casas decimais (10,-5,0,1000).
•double = Números com casas decimais (10.5; 3.14; -7.89).
•boolean = Valores lógicos verdadeiro ou falso (true ou false).
•char = Um único caractere ('A', 'z', '5', '@').
•String = Sequência de caracteres ( "Maikão", "Olá Mundo!").

Estruturas de controle: Tomando Decisões

 Para que o programa tome decisões baseadas em condições, usamos as estruturas if(se) e else(senão). Elas permitem que seu código execute diferentes ações dependendo das circunstâncias.

Exemplo prático: Verificando maioridade

 public class Decisao {
   public static void main(String[] args) {
    int idade = 17

    if(idade >= 18){
       System.out.println("Acesso permitido.");
      }else {
        System.out.println("Acesso negado.");
        System.out.println("Menor de IDADE.");
     }
   }
 }

Estruturas de Repetição: Loops

 Por que usar loops?
  Quando você precisarepetir uma ação várias vezes, loops tornam seu código mais efiviente e limpo. Em vez de escrever o mesmo comando repetidamente, você define uma estrutura de repetição.

 O Loop for
  O loop mais comum em Java. Ele tem três partes:
 
  Inicialização: onde começa a contagem
  Condição: até quando repetir
  Incremento: como avançar a cada vez

Exemplo: Contando de 1 a 5

 public class Loop{
   public static void main(String[] args){
    //(início; condição de parada; incremento)
    for(int i = 1; i <= 5; i++){
      System.out.println("Contagem:" +i);
    }
  }
}
